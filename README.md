# study-note
### 树
#### 二叉树
##### 定义及基本术语
根，左右子树，空树，结点（根结点，分支结点，叶子结点），结点度数(0,1,2)。  
层数，深度（自顶向下），高度（自底向上）。  
满二叉树，完全二叉树（n个结点与满二叉树前n个结点一一对应）。  
##### 基本属性
1.第i层最多2<sup>i-1</sup>个结点。  
2.深度为k，最多2<sup>k</sup>-1个结点。  
3.度为0节点数n<sub>0</sub>，n<sub>0</sub>=n<sub>2</sub>-1。  
4.n个结点的完全二叉树，深度为log<sub>2</sub>n下取整+1。
